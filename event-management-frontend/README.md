# EventHub - Frontend

Modern event management platform frontend built with React, Tailwind CSS, and shadcn/ui.

## Setup

1. Clone the repo
2. Run `npm install`
3. Create `.env` file with `REACT_APP_API_URL`
4. Run `npm start`

## Deployment

Build with `npm run build` and deploy the `build` folder to Netlify.

## Features

- JWT authentication with role guards
- Full CRUD for events (multi-step wizard)
- Attendee management with check-in
- Dashboard with charts and activity feed
- Dark mode and responsive design
